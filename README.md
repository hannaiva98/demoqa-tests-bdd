# DemoQA Tests BDD

Автоматизированные тесты для сайта DemoQA с использованием Cypress и BDD (Cucumber).

## Технологии

- Cypress 14.x
- Cucumber (via @badeball/cypress-cucumber-preprocessor)
- JavaScript
- GitHub Actions (CI)

## Установка

```bash
npm install
```
